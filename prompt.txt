Build a simple wheel spinner web app using React, Vite, TailwindCSS, and shadcn/ui. The app should include a textarea where users can enter newline-separated options, which are rendered as segments of a spinning wheel. Provide a button to spin the wheel; the animation should start fast and gradually slow down until it lands on a randomly selected option, with uniform randomness across all entries. On mobile devices, users should also be able to spin the wheel by swiping or dragging the wheel with their finger. Display the selected result clearly after each spin, include a test that simulates many spins (e.g., ~10,000) to verify the fairness of the selection distribution, and make the state shareable via URL parameters so the current options can be shared with a link.
